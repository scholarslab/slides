GIS Workshop: Learning Old-School Mapping Techniques

Wednesday, March 4
1:00 - 2:00pm
Alderman Library, Room 421

How did folks make maps before GPS and satellite imagery? In this workshop we’ll focus on plane table mapping. Using just a flat surface, a sheet of paper, a straight edge, and a pencil we’ll learn techniques to create accurate maps for large geographic areas. With plane table mapping, if you can see it, you can map it. This technique can be particularly helpful to researchers such a biologists working in small areas under heavy tree canopy where GPS fails. Instructor: Chris Gist
